# react-portfolio-page
Rebuilding my portfolio page in ReactJS using Webpack and Babel as dev tools.
## To install
- `git init`
- `git clone https://github.com/korrollir/react-portfolio-page.git`
- `npm install`
- `npm start`
- open browser and go to `localhost:8080`
